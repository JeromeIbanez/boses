## Common patterns

### Pagination
List endpoints return arrays. No cursor-based pagination currently — all records for the authenticated company are returned.

### IDs
All resource IDs are UUIDs (e.g. `"3fa85f64-5717-4562-b3fc-2c963f66afa6"`).

### Timestamps
All timestamps are UTC ISO 8601 strings: `"2026-04-07T08:30:00"`.

### Errors

```json
{
  "detail": "Simulation not found"
}
```

Standard HTTP status codes: `400` bad request, `401` unauthenticated, `403` forbidden, `404` not found, `422` validation error, `500` server error.
